# wa-migrate

## How run

1. Change the name of data.json.dev to data.json
4. Set your data in the data.json
5. Run with `go run .`

## Examples

data.json
```json
[
    {
        "url": "https://foo.bar",
        "filename": "bar.json",
        "username": "admin",
        "password": "my_password1"
    },
    {
        "url": "https://foo.bar2",
        "filename": "bar2.json",
        "username": "admin",
        "password": "my_password2"
    }
]
```
